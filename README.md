# FitTrack

Application de suivi d'entraînement et de nutrition.

## 🚀 Application en production

**https://fit-track-iv5a.vercel.app**

## Architecture

| Couche | Service | URL |
|---|---|---|
| Frontend | Vercel | https://fit-track-iv5a.vercel.app |
| Backend | Render | https://fittrack-backend-i3wz.onrender.com |
| Base de données | Clever Cloud | MySQL DEV |

## Stack technique

- **Frontend** : React + TypeScript + Vite
- **Backend** : Node.js + Express (Docker)
- **Base de données** : MySQL

## Lancer en local

```bash
docker compose up
```

L'application est disponible sur http://localhost:3000
